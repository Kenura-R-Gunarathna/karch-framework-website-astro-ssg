# KARCH Framework Documentation

This is the official documentation for the KARCH Framework, built with [Astro Starlight](https://starlight.astro.build/).

## Getting Started

1.  **Install dependencies:**
    ```bash
    npm install
    ```

2.  **Start the development server:**
    ```bash
    npm run dev
    ```

3.  **Build for production:**
    ```bash
    npm run build
    ```
