A light-weight and fast MVC framework for php developers.
This is build with a system similar to Laravel. Developers
who have experience on Laravel can easily adopt to uour light-weight framework.

##### Github Code Example:

You can refer and star our repository on [Github](https://github.com/Kenura-R-Gunarathna/KARCH-Framework) as well.

##### Requirements:

- **PHP:** `version >= 8.0`
